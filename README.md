## How to install
   - Multiline command:
      * git clone https://github.com/Zulfilham/Z-TCFG;
      * bash Z-TCFG/install.sh;

   - Oneline command:
      * git clone https://github.com/Zulfilham/Z-TCFG && bash Z-TCFG/install.sh;
